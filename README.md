# 📈 Calculus Tangent Line Visualizer

A Python project that visually demonstrates the concept of derivatives and tangent lines in calculus using NumPy and Matplotlib.

## ✨ Features

- Plot mathematical functions
- Visualize tangent lines
- Display point of tangency
- Calculate derivative values
- Show slope of the tangent
- Display tangent line equation
- Shade area under the curve
- Add annotations and labels
- Educational calculus visualization

## 📊 Example

The project visualizes:

f(x) = x²

Derivative:

f'(x) = 2x

At x = -5:

- Point of tangency: (-5, 25)
- Slope: -10
- Tangent line:
  
  y = -10x - 25

## 🛠️ Technologies Used

- Python
- NumPy
- Matplotlib
